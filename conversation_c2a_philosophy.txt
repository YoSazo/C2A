4. THE ACTIVE LESSON SYSTEM (Implementation)
--------------------------------------------
Problem: Users would get feedback ("You're bridge coping") but forget it by
next session. Learning loop was open.

Solution: Active Lesson System
- Each session's growth_edge becomes an "Active Lesson"
- Lesson is displayed prominently before each session
- Judge checks if lesson was applied
- Lesson persists until demonstrated (score 70+ AND applied)
- Then new lesson from that session's growth_edge

This closes the learning loop:
Feedback → Practice → Verification → Next skill
